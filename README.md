# راهنمای نصب و کامپایل برنامه Island Overlay

## پیش‌نیازها
1. **نرم‌افزارهای مورد نیاز**:
   - Java Development Kit (JDK) نسخه 8 یا بالاتر
   - Android Studio
   - Gradle

2. **سیستم‌عامل**:
   - Windows, macOS یا Linux

## مراحل نصب

### 1. گیت و مخزن
   - ابتدا اطمینان حاصل کنید که Git بر روی سیستم شما نصب شده باشد.
   - سپس، مخزن پروژه را با استفاده از دستور زیر کلون کنید:
     ```bash
     git clone https://github.com/Alireza-Crdroid/Noi.git
     ```

### 2. باز کردن پروژه در Android Studio
   - Android Studio را باز کرده و گزینه **Open an existing Android Studio project** را انتخاب کنید.
   - مسیر پروژه را که کلون کرده‌اید، انتخاب کنید.

### 3. پیکربندی Gradle
   - پس از باز کردن پروژه، Gradle به‌طور خودکار بارگذاری خواهد شد. ممکن است برای بارگذاری تمام وابستگی‌ها زمان بگیرد.

### 4. کامپایل و اجرای برنامه
   - بر روی دکمه **Run** در Android Studio کلیک کنید.
   - اطمینان حاصل کنید که یک شبیه‌ساز یا دستگاه واقع�� به سیستم متصل است.

## راه‌اندازی برنامه
- پس از اجرای موفقیت‌آمیز برنامه، می‌توانید آن را از لیست برنامه‌ها در دستگاه خود پیدا کنید.

## نکات
- در صورت بروز خطا، از تب **Build** در Android Studio برای بررسی و رفع مشکلات استفاده کنید.
